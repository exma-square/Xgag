# Xgag

Express.js starter, collection for most usage library.

## pre-install

 * node.js@0.10.22
 * npm
 * [mongojs](https://github.com/mafintosh/mongojs)
 * [express@4.11.1rc](http://github.com/strongloop/express/)

## how to start

```
npm start
open http://localhost:7000
```
